1. Datenschutz auf einen Blick

Allgemeine Hinweise

Die folgenden Hinweise geben einen einfachen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn Sie diese Website besuchen. Personenbezogene Daten sind alle Daten, mit denen Sie persönlich identifiziert werden können.

Demo-Hinweis

Wichtig: Dies ist eine Demo-Website. Alle Daten werden ausschließlich lokal in Ihrem Browser (localStorage) gespeichert und niemals an einen Server übertragen. Es findet keine tatsächliche Datenverarbeitung oder -speicherung auf einem Server statt.

2. Datenerfassung auf dieser Website

Wer ist verantwortlich für die Datenerfassung?

Die Datenverarbeitung auf dieser Demo-Website erfolgt ausschließlich lokal in Ihrem Browser. Es werden keine Daten an externe Server übermittelt.

Wie erfassen wir Ihre Daten?

In dieser Demo-Anwendung werden folgende Daten lokal in Ihrem Browser gespeichert:

  • Anmeldedaten (E-Mail, Name, verschlüsseltes Passwort)
  • Buchungsinformationen
  • Cookie-Einstellungen

Diese Daten werden nur in Ihrem Browser gespeichert und verlassen niemals Ihren Computer.

3. Allgemeine Hinweise und Pflichtinformationen

Datenschutz

Die Betreiber dieser Demo-Seite nehmen den Schutz Ihrer persönlichen Daten sehr ernst. Diese Demo-Anwendung verarbeitet keinerlei echte personenbezogene Daten auf einem Server.

4. Cookies

Diese Website verwendet sogenannte "Cookies" und localStorage. Cookies sind kleine Textdateien, die auf Ihrem Endgerät gespeichert werden. localStorage ist ein Browser-Feature zur lokalen Datenspeicherung.

In dieser Demo verwenden wir:

  • Cookie-Consent: Speichert Ihre Cookie-Einwilligung
  • Session-Daten: Speichert Ihren Login-Status
  • Anwendungsdaten: Fahrzeuge, Buchungen (nur Demo-Daten)

5. Ihre Rechte

Da diese Demo-Anwendung keine Daten auf einem Server speichert, können Sie alle Ihre Daten jederzeit selbst löschen, indem Sie:

  • Den localStorage Ihres Browsers löschen
  • Die Cookies dieser Website löschen
  • Die Browser-Konsole öffnen und localStorage.clear() ausführen

6. Analyse-Tools und Tools von Drittanbietern

Diese Demo-Website verwendet keine Analyse-Tools oder Tracking-Software von Drittanbietern.

7. Bootstrap und FontAwesome

Diese Website nutzt Bootstrap CSS und FontAwesome Icons, die von CDN-Servern geladen werden:

  • Bootstrap CDN (cdn.jsdelivr.net)
  • FontAwesome CDN (cdnjs.cloudflare.com)

Beim Laden dieser Ressourcen kann Ihre IP-Adresse an diese CDN-Anbieter übermittelt werden.

8. Bilder

Die auf dieser Website verwendeten Fahrzeugbilder stammen von Unsplash (images.unsplash.com). Beim Laden dieser Bilder kann Ihre IP-Adresse an Unsplash übermittelt werden.
